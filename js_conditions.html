JavaScript if else ve else if


Koşullu ifadeler, farklı koşullara bağlı olarak farklı eylemler gerçekleştirmek için kullanılır.

Koşullu İfadeler


Çoğu zaman kod yazarken, farklı kararlar için farklı eylemler gerçekleştirmek istersiniz.
Bunu yapmak için kodunuzda koşullu ifadeler kullanabilirsiniz.
JavaScript'te şu koşullu ifadelere sahibiz:
  • Belirli bir koşul doğruysa yürütülecek bir kod bloğu belirtmek için if kullanın
  • Aynı koşul yanlışsa yürütülecek bir kod bloğu belirtmek için else kullanın
  • Test edilecek yeni bir koşul belirtmek için, ilk koşul yanlışsa, else if kullanın
  • Yürütülecek birçok kod bloğunu belirtmek için switch'i kullanın

    • If koşulu


    Bir koşul doğruysa yürütülecek JavaScript kod bloğunu belirtmek için if ifadesini kullanın.
    Syntax 
        if (condition) {
      //  koşul doğruysa yürütülecek kod bloğu
    }

    Örnek
    Saat 18: 00'den geriyse "İyi günler" selamlaması yapın: 
        if (hour < 18) {
      greeting = "Good day";
    }

    Sonuç : Good day

    Else koşulu


    Koşul i olursa yürütülecek bir kod bloğu belirtmek için else deyimini kullanın
    Syntax 
        if (condition) {
      // koşul doğruysa yürütülecek kod bloğu
    } else {
      //  koşul yanlışsa yürütülecek kod bloğu
    }

    Örnek
    Saat 18'den geriyse, "İyi günler" selamı, aksi halde "İyi akşamlar" oluşturun: 
        if (hour < 18) {
      greeting = "Good day";
    } else {
      greeting = "Good evening";
    }

    Else if koşulu


    İlk koşul yanlışsa yeni bir koşul belirtmek için else if deyimini kullanın.
    Syntax 
        if (condition1) {
      //  koşul1 doğruysa yürütülecek kod bloğu
    } else if (condition2) {
      //  koşul1 yanlış ve koşul2 doğruysa yürütülecek kod bloğu
    } else {
      //  koşul1 yanlış ve koşul2 yanlışsa yürütülecek kod bloğu
    }

    Örnek
    Saat 10:00'dan geriyse, "Günaydın" karşılaması oluşturun, değilse zaman 20:00'den geriyse, "İyi günler" karşılaması oluşturun, aksi takdirde "İyi akşamlar": 
        if (time < 10) {
      greeting = "Good morning";
    } else if (time < 20) {
      greeting = "Good day";
    } else {
      greeting = "Good evening";
    }
