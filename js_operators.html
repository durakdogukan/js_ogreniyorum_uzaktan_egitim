JavaScript Operatörleri


Değişkenlere değerler atayın ve bunları birlikte ekleyin: 
    var x = 5;         // x değişkenine 5 atandı
    var y = 2;         // y değişkenine 2 atandı
    var z = x + y;     // z değişkenine 7 atandı (x + y)

    var x = 10;

Operatör Örnek Benzer

Toplama atama işleci (+ =) bir değişkene değer ekler.

JavaScript String Operatörleri


+ Operatörü, stringleri eklemek (birleştirmek) için de kullanılabilir. 
    var txt1 = "John";
    var txt2 = "Doe";
    var txt3 = txt1 + " " + txt2;

    Txt3 sonucu şöyle olur: John Doe

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


+ = Atama operatörü, stringleri eklemek (birleştirmek) için de kullanılabilir: 
    var txt1 = "What a very ";
    txt1 += "nice day";

    txt1 sonucu şöyle olur: What a very nice day

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Dize ve Sayı Ekleme


İki sayı eklemek toplamı döndürür, ancak sayı ve dize eklemek bir dize döndürür: 
    var x = 5 + 5;
    var y = "5" + 5;
    var z = "Hello" + 5;


    Burada beklediğimiz sonuçlar:
    10
    55
    Hello5

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.
