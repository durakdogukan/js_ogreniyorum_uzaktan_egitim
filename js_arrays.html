JavaScript Diziler


JavaScript dizileri, birden çok değeri tek bir değişkende depolamak için kullanılır. 
    var cars = ["Saab", "Volvo", "BMW"];

Dizi nedir ?



Dizi, aynı anda birden fazla değeri tutabilen özel bir değişkendir.
Bir öğe listeniz varsa (örneğin, araba isimleri listesi), arabaları tek değişkenlerde saklamak şöyle görünebilir: 
    var car1 = "Saab";
    var car2 = "Volvo";
    var car3 = "BMW";

Ancak, ya arabalar arasında dolaşıp belirli bir tane bulmak istiyorsanız? Peki ya 3 değil, 300 arabanız olsaydı?
Çözüm bir dizi!
Bir dizi, tek bir ad altında birçok değeri tutabilir ve değerlere bir dizin numarasına başvurarak erişebilirsiniz.

Dizi oluşturma


Bir dizi değişmezi kullanmak, bir JavaScript Dizisi oluşturmanın en kolay yoludur.
Syntax: 
    var array_name = [item1, item2, ...];

 
  
    var cars = ["Saab", "Volvo", "BMW"];

Dizi Öğeleri Nesne Olabilir


JavaScript değişkenleri nesne olabilir. Diziler özel nesnelerdir.
Bu nedenle, aynı Dizide farklı türde değişkenlere sahip olabilirsiniz.
Bir Dizide nesneler olabilir. Bir Dizide işlevleriniz olabilir. Bir Dizide diziler olabilir: 
    myArray[0] = Date.now;
    myArray[1] = myFunction;
    myArray[2] = myCars;

Dizi Özellikleri ve Yöntemleri


JavaScript dizilerinin gerçek gücü yerleşik dizi özellikleri ve yöntemleridir:

    var y = cars.sort();   // Sort () yöntemi dizileri sıralar

Döngü Dizisi Elemanları


Bir dizi boyunca döngü oluşturmanın en güvenli yolu bir for döngüsü kullanmaktır: 
        
 var fruits, text, fLen, i;
fruits = ["Banana", "Orange", "Apple", "Mango"];

  fLen = fruits.length;

text = "<ul>";
for (i = 0; i < fLen; i++) {
  text += "<li>" + fruits[i] + "</li>";
}
text 
  += "</ul>";

İlişkilendirilebilir Diziler


Birçok programlama dili, adlandırılmış dizinlere sahip dizileri destekler.
Adlandırılmış dizinleri olan dizilere ilişkilendirilebilir diziler denir.
JavaScript, adlandırılmış dizine sahip dizileri desteklemez.
JavaScript'te, diziler her zaman numaralı dizinler kullanır. 
    var person = [];
    person[0] = "John";
    person[1] = "Doe";
    person[2] = 46;
    var x = person.length;     // person.length 3 döndürür
    var y = person[0];         // person[0] döndürür

Diziler ve Nesneler Arasındaki Fark


JavaScript'te, diziler numaralı dizinler kullanır.
JavaScript'te nesneler adlandırılmış dizinler kullanır.

Diziler Ne Zaman Kullanılmalı. Nesneler ne zaman kullanılır?


  • JavaScript ilişkilendirilebilir dizileri desteklemez.
  • Eleman adlarının dize (metin) olmasını istediğinizde nesneleri kullanmalısınız.
  • Eleman adlarının sayı olmasını istediğinizde diziler kullanmalısınız.

    • Yeni Diziden Kaçının ()


    JavaScript'in yerleşik dizi yapıcısı yeni Array'ı () kullanmaya gerek yoktur.
    Bunun yerine [] kullanın.
    Bu iki farklı ifadenin her ikisi de points adlı yeni bir boş dizi oluşturur: 
        var points = new Array();     // Kötü
    var points = [];              // İyi

    Bu iki farklı ifadenin her ikisi de 6 sayı içeren yeni bir dizi oluşturur: 
        var points = new Array(40, 100, 1, 5, 25, 10); // Kötü
    var points = [40, 100, 1, 5, 25, 10];          // İyi

    Bir Diziyi Tanıma


    Yaygın bir soru şudur: Bir değişkenin dizi olup olmadığını nasıl anlarım?
    Sorun JavaScript operatörü typeof "object" döndürüyor olmasıdır: 
        var fruits = ["Banana", "Orange", "Apple", "Mango"];

    typeof fruits;    // object döndürür

    Örneği incelemek için tıklayın
     
     

    NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


    JavaScript dizisi bir nesne olduğu için typeof operatörü nesneyi döndürür.
    Çözüm 1:
    Bu sorunu çözmek için ECMAScript 5 yeni bir yöntem Array.isArray () tanımlar: 
    Array.isArray(fruits);   // true döndürür

    Örneği incelemek için tıklayın
     
     

    NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


    Bu çözümün sorunu ECMAScript 5'in eski tarayıcılarda desteklenmemesidir.
    Çözüm 2:
    Bu sorunu çözmek için kendi isArray() fonksiyonunu oluşturabilirsiniz: 
        function isArray(x) {
      return x.constructor.toString().indexOf("Array") > -1;
    }

    Örneği incelemek için tıklayın
     
     

    NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


    Bağımsız değişken bir dizi ise yukarıdaki fonksiyon her zaman true değerini döndürür.
    Daha doğrusu: nesne prototipi "Dizi" kelimesini içeriyorsa true değerini döndürür.
    Çözüm 3:
    İnstanceof operatörü, belirli bir kurucu tarafından bir nesne oluşturulursa true değerini döndürür: 
        var fruits = ["Banana", "Orange", "Apple", "Mango"];
    fruits instanceof Array;   // true döndürür
