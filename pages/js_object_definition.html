JavaScript Nesneler


  • JavaScript'te neredeyse "her şey" bir nesnedir.
  • Boolean'lar nesne olabilir (new anahtar kelime ile tanımlanmışsa)
  • Sayılar nesne olabilir (new anahtar kelime ile tanımlanmışsa)
  • Dizeler nesne olabilir (new anahtar kelime ile tanımlanmışsa)
  • Tarihler her zaman nesnedir
  • Matematik her zaman nesnedir
  • Normal ifadeler her zaman nesnedir
  • Diziler her zaman nesnedir
  • Fonksiyonlar her zaman nesnedir
  • Nesneler her zaman nesnedir
  • Temel öğeler dışındaki tüm JavaScript değerleri nesnedir.

    • JavaScript Temel Öğeleri


    İlkel değer, özelliği veya yöntemi olmayan bir değerdir.
    İlkel veri türü, ilkel değere sahip verilerdir.
    JavaScript 5 tür ilkel veri türü tanımlar:
  • string
  • number
  • boolean
  • null
  • undefined

    • İlkel değerler değişmezdir (sabit kodlanmıştır ve bu nedenle değiştirilemezler).
    Değer Tip Açıklama
    "Hello" string "Hello" is always "Hello"
    3.14 number 3.14 is always 3.14
    true boolean true is always true
    false boolean false is always false
    null null (object) null is always null
    undefined undefined undefined is always undefined

    JavaScript nesnesi, adlandırılmış değerlerin bir koleksiyonudur

    Nesne Özellikleri


    JavaScript nesnelerindeki adlandırılmış değerlere özellikler denir.
    Özellik Değer
    firstName John
    lastName Doe
    age 50
    eyeColor blue

    Ad değeri çifti olarak yazılan nesneler şuna benzer:
  • PHP'de ilişkilendirilebilir diziler
  • Python'da Dictionaries
  • C'deki Hash tables
  • Java'daki Hash maps
  • Ruby ve Perl'de Hashes

    • Nesne Yöntemleri


    Yöntemler nesneler üzerinde gerçekleştirilebilecek eylemlerdir .
    Nesne özellikleri hem ilkel değerler, diğer nesneler hem de fonksiyonlar olabilir.
    Özellik Değer
    firstName John
    lastName Doe
    age 50
    eyeColor blue
    fullName function() {return this.firstName + " " + this.lastName;}

    JavaScript nesneleri, adlandırılmış değerler için özellikler ve yöntemler adı verilen kaplardır.

    JavaScript Nesnesi Oluşturma


    JavaScript ile kendi nesnelerinizi tanımlayabilir ve oluşturabilirsiniz.
    Yeni nesneler oluşturmanın farklı yolları vardır:
  • Bir nesne hazır bilgisi kullanarak tek bir nesne tanımlayın ve oluşturun.
  • new anahtar kelimes, ile tek bir nesne tanımlayın ve oluşturun .
  • Bir nesne yapıcı tanımlayın ve sonra da inşa edilen türde nesneler oluşturun.

    • ECMAScript 5'te fonksiyonla bir nesne de oluşturulabilir Object.create().

    JavaScript Nesne Değişmezi Kullanma


    JavaScript Nesnesi oluşturmanın en kolay yolu budur.
    Bir nesne değişmezi kullanarak, bir deyimde bir nesne tanımlar ve yaratırsınız.
    Nesne değişmez değeri küme parantezlerinin içindeki {} ad çiftleri (yaş: 50 gibi) listesidir.
    Aşağıdaki örnek, dört özelliğe sahip yeni bir JavaScript nesnesi oluşturur:
    Örnek 
        var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};

    JavaScript Nesneleri Değişebilir


    Nesneler değiştirilebilir: Değer ille değil referansla adreslenir.
    Kişi bir nesneyse, aşağıdaki ifade kişinin bir kopyasını oluşturmaz: 
        var x = person;  // Bu kişinin bir kopyasını oluşturmaz.

    X nesnesi kişinin bir kopyası değildir. Hem x hem de kişi aynı nesnedir.
    X ile kişi aynı nesne olduğu için x üzerinde yapılan değişiklikler de kişiyi değiştirir.
    Örnek: 
        var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"}

    var x = person;
    x.age = 10;           // Bu hem x.age hem de person.age değişkenlerini değiştirir
