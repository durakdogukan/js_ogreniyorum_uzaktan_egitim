JavaScript Derinlemesine Fonksiyon Çağırma


Bir JavaScript fonksiyonun içindeki kod, "bir şey" çağırdığında çalıştırılır.

JavaScript Fonksiyonunu Çağırma


Fonksiyon tanımlandığında fonksiyon içindeki kod yürütülmez.
Bir fonksiyonun içindeki kod, fonksiyon çağrıldığında yürütülür.
"Bir fonksiyonu çağırmak" yerine "bir fonksiyonu çağır" terimini kullanmak yaygındır.
"Bir fonksiyonu çağırmak", "bir fonksiyonu başlatmak" veya "bir fonksiyonu yürütmek" demek yaygındır.
Bu bölümde, invoke kullanacağız, çünkü bir JavaScript fonksiyonu çağrılmadan çağrılabilir.

Bir Fonksiyonu Fonksiyon Olarak Çağırma


Örnek: 
    function myFunction(a, b) {
      return a * b;
    }
    myFunction(10, 2);           // 20 sonucunu döndürür

Yukarıdaki fonksiyon herhangi bir nesneye ait değildir. Ancak JavaScript'te her zaman varsayılan bir global nesne vardır.
HTML'de varsayılan global nesne HTML sayfasının kendisidir, bu nedenle yukarıdaki fonksiyon HTML sayfasına "aittir".
Bir tarayıcıda sayfa nesnesi tarayıcı penceresidir. Yukarıdaki fonksiyon otomatik olarak bir pencere fonksiyonu haline gelir.
myFunction () ve window.myFunction () aynı işlevdir:
Örnek: 
    function myFunction(a, b) {
      return a * b;
    }
    window.myFunction(10, 2);    // 20 sonucunu döndürür

This Anahtarı


JavaScript olarak adlandırılan şey, geçerli koda "sahip" olan nesnedir.
Bunun değeri, bir fonksiyonda kullanıldığında, fonksiyonun "sahibi" olan nesnedir.

Global Nesne


Bir fonksiyon sahip nesnesi olmadan çağrıldığında, bunun değeri global nesne haline gelir.
Bir web tarayıcısında global nesne tarayıcı penceresidir.
Bu örnek, pencere nesnesini bunun değeri olarak döndürür:
Örnek: 
    var x = myFunction();            // x window object

    function myFunction() {
      return this;
    }

Bir Fonksiyonu Yöntem Olarak Çağırma


JavaScript'te fonksiyonları nesne yöntemleri olarak tanımlayabilirsiniz.
Aşağıdaki örnek, iki özellik (firstName ve lastName) ve bir yöntem (fullName) içeren bir nesne (myObject) oluşturur:
Örnek: 
    var myObject = {
      firstName:"John",
      lastName: "Doe",
      fullName: function () {
        return this.firstName + " " + this.lastName;
      }
    }
    myObject.fullName();         // Will return "John Doe"

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


FullName yöntemi bir fonksiyondur. Fonksiyon nesneye aittir. myObject, fonksiyonun sahibidir.
Buna denilen şey, JavaScript kodunun "sahibi" olan nesnedir. Bu durumda bunun değeri myObject olur.
Bunun değerini döndürmek için fullName yöntemini değiştirin:
Örnek: 
    var myObject = {
      firstName:"John",
      lastName: "Doe",
      fullName: function () {
        return this;
      }
    }
    myObject.fullName();          // [object Object] döndürecek (the owner object)

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Fonksiyon Oluşturucu ile Fonksiyon Çağırma


Bir fonksiyon çağırma işleminden önce new anahtar sözcük gelirse, bu bir yapıcı çağırma işlemidir.
Yeni bir fonksiyon oluşturduğunuz anlaşılıyor, ancak JavaScript fonksiyonları nesneler olduğu için aslında yeni bir nesne oluşturuyorsunuz:
Örnek: 
    // Function constructor:
    function myFunction(arg1, arg2) {
      this.firstName = arg1;
      this.lastName  = arg2;
    }

    // Yeni nesne oluşturuluyor
    var x = new myFunction("John", "Doe");
    x.firstName;                             // "John" döndürür

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Bir yapıcı çağırma yeni bir nesne oluşturur. Yeni nesne özellikleri ve yöntemleri yapıcısından devralır.
