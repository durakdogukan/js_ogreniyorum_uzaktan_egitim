JavaScript Escape Karakter

Stringler tırnak içinde yazılması gerektiğinden, JavaScript bu dizeyi yanlış anlayacaktır: 
    var x = "We are the so-called "Vikings" from the north.";

String "Biz sözde" olarak kesilecektir.
Bu problemden kaçınmanın çözümü ters eğik çizgi escape karakterini kullanmaktır.
Ters eğik çizgi (\) çıkış karakteri özel karakterleri string karakterlerine dönüştürür:
Kod Sonuç Açıklama
\' ' Tek Tırnak
\" " Çift Tırnak
\\ \ Ters Slash

\" Stringi bir dizeye çift tırnak ekler: 
    var x = "We are the so-called \"Vikings\" from the north.";

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


\' Stringi bir dizeye çift tırnak ekler: 
    var x = "We are the so-called \'Vikings\' from the north.";

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


\\ string'i bir dizeye ters eğik çizgi ekler: 
    var x = "The character \\ is called backslash.";

JavaScript'te altı başka escape string'i daha geçerlidir:
Kod Sonuç
\b Backspace
\f Form Feed
\n Yeni Satır
\r Satırbaşı
\t Yatay Tablo
\v Dikey Tablo

Stringler Dizeler Nesne Olabilir


Normalde, JavaScript stringleri değişmez değerlerden oluşturulan ilkel değerlerdir:
var firstName = "John";
Ancak stringler, new anahtar kelimesine sahip nesneler olarak da tanımlanabilir:
var firstName = new String("John"); 
    var x = "John";
    var y = new String("John");

    // typeof x String döndürecek.
    // typeof y Object döndürecek.

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


== işlecini kullanırken, eşit stringler eşittir: 
    var x = "John";
    var y = new String("John");

    // (x == y) Doğrudur çünkü x ve y değerleri birbirine eşittir.

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


=== işleci kullanılırken eşit stringler eşit değildir, çünkü === işleci hem tür hem de değerde eşitlik bekler. 
    var x = "John";
    var y = new String("John");

    // (x === y) yanlış çünkü x ve y farklı tiplerdedir (string ve object)

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Veya daha da kötüsü. Nesneler karşılaştırılamaz: 
    var x = new String("John");
    var y = new String("John");

    // (x == y) yanlış çünkü x ve y farklı objeler

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz. 


    var x = new String("John");
    var y = new String("John");

    // (x === y) is false because x and y are different objects

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


(X == y) ve (x === y) arasındaki farka dikkat edin.
İki JavaScript nesnesinin karşılaştırılması her zaman yanlış döndürür.
