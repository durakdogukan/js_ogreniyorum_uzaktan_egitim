JavaScript Sayılar


JavaScript'in yalnızca bir tür sayı tipi vardır. Sayılar ondalıklı veya ondalıksız olarak yazılabilir. 
    var x = 3.14;    // Ondalıklı şekilde yazılmış bir sayı
    var y = 3;       // Ondalıklı şekilde yazılmamış bir sayı

JavaScript'te Sayılar Her Zaman 64 bit Kayan Noktadır


Diğer birçok programlama dilinin aksine, JavaScript tamsayılar, kısa, uzun, kayan nokta vb.Gibi farklı sayı türlerini tanımlamaz.
JavaScript numaraları her zaman uluslararası IEEE 754 standardını izleyerek çift kesinlikli kayar nokta sayısı olarak saklanır.
Bu biçim sayıları 64 bitte saklar; burada sayı (kesir) 0 ila 51 bitlerinde, üs 52 ila 62 bitlerinde ve oturum açma biti 63'te saklanır:
Değer Üs İşaret
52 bits (0 - 51) 11 bits (52 - 62) 1 bit (63)

Hassasiyet

Tamsayılar (nokta veya üs notasyonu olmayan sayılar) 15 basamağa kadar doğrudur. 
    var x = 999999999999999;   // x will be 999999999999999
    var y = 9999999999999999;  // y will be 10000000000000000

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Maksimum ondalık sayı 17'dir, ancak kayan nokta aritmetiği her zaman % 100 doğru değildir: 
    var x = 0.2 + 0.1;         // x 0.30000000000000004 olacak

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Yukarıdaki sorunu çözmek için çarpma ve bölmeye yardımcı olur: 
    var x = (0.2 * 10 + 0.1 * 10) / 10;       // x 0.3 olacak.

Sayı ve String Ekleme


UYARI !!
JavaScript hem ekleme hem de birleştirme için + işlecini kullanır.
Sayılar eklenir. Dizeler birleştirilir.

İki sayı eklerseniz sonuç bir sayı olur: 
    var x = 10;
    var y = 20;
    var z = x + y;           // z değeri 30 olacak (sayı)

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


İki string eklerseniz, sonuç bir string birleşimi olur: 
    var x = "10";
    var y = "20";
    var z = x + y;           // z değeri 1020 olacak (string)

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Bir sayı ve string eklerseniz, sonuç bir string birleşimi olur: 
    var x = 10;
    var y = "20";
    var z = x + y;           // z değeri 1020 olacak (string)

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Bir string ve sayı eklerseniz, sonuç bir string birleştirmesi olur: 
    var x = "10";
    var y = 20;
    var z = x + y;           // z değeri 1020 olacak (string)

Örneği incelemek için tıklayın

Yaygın bir hata, bu sonucun 30 olmasını beklemektir:

    var y = 20;
    var z = "The result is: " + x + y;

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Yaygın bir hata, bu sonucun 102030 olmasını beklemektir:

Daha sonra z + bir string olduğu için 30 + "30" birleştirilir.

NaN - Not a Number


NaN, bir sayının yasal bir sayı olmadığını belirten JavaScript'e ayrılmış bir kelimedir.
Sayısal olmayan bir dize ile aritmetik yapmaya çalışmak NaN (Sayı Değil) ile sonuçlanır: 
    var x = 100 / "Apple";  // x NaN olacak (Sayı değil)

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Ancak, string sayısal bir değer içeriyorsa sonuç bir sayı olur: 
    var x = 100 / "10";     // x 10 olacak

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Bir değerin sayı olup olmadığını öğrenmek için isNaN () genel JavaScript fonksiyonunu kullanabilirsiniz: 
    var x = 100 / "Apple";
    isNaN(x);               // True döndürecek çünkü x bir NaN

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


NaN'a dikkat et. Bir matematik işleminde NaN kullanırsanız, sonuç NaN olur:

    var y = 5;
    var z = x + y;         // z NaN olacak

Veya sonuç bir birleştirme olabilir: 
    var x = NaN;
    var y = "5";

NaN bir sayıdır: typeof NaN sayı döndürür:

Infinity


Sonsuz (veya -Infinity), mümkün olan en büyük sayının dışında bir sayı hesaplarsanız JavaScript'in döndüreceği değerdir. 
    var myNumber = 2;
    while (myNumber != Infinity) {   // Sonsuza kadar çalıştır.
    myNumber = myNumber * myNumber;
}

0'a (sıfır) bölme de Sonsuzluk üretir: 
    var x =  2 / 0;       // x +Sonsuz
    var y = -2 / 0;       // y -Sonsuz

Sayılar Nesne Olabilir


Normalde JavaScript sayıları değişmez değerlerden oluşturulan ilkel değerlerdir:
var x = 123;
Ancak sayılar, yeni anahtar kelimeye sahip nesneler olarak da tanımlanabilir:
var y = new Number(123); 
    var x = 123;
    var y = new Number(123);

    // typeof x number döndürecek
    // typeof y object döndürecek

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


== operatörü kullanırken eşit sayılar eşittir: 
    var x = 500;
    var y = new Number(500);

    // (x == y) true döndürür çünkü x ve y değerleri eşittir

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


=== operatörü kullanılırken eşit sayılar eşit değildir, çünkü === işleci hem tür hem de değerde eşitlik bekler. 
    var x = 500;
    var y = new Number(500);

    // (x === y) false döndürür çünkü x ve y farklı tiplerdeler

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Veya daha da kötüsü. Nesneler karşılaştırılamaz: 
    var x = new Number(500);
    var y = new Number(500);

    // (x == y) false döndürür çünkü nesneler karşılaştırılamaz

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.
