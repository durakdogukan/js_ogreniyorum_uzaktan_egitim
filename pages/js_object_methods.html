JavaScript Nesne Yöntemleri


Örnek: 
    var person = {
      firstName: "John",
      lastName : "Doe",
      id       : 5566,
      fullName : function() {
        return this.firstName + " " + this.lastName;
      }
    };

This Anahtarı


Fonksiyon tanımında, fonksiyonun this"sahibine" karşılık gelir.
this.firstName bu nesnenin firstName özelliği anlamına gelir .

JavaScript Methods


JavaScript yöntemleri, nesneler üzerinde gerçekleştirilebilecek eylemlerdir.
JavaScript yöntemi , fonksiyon tanımı içeren bir özelliktir .
Property Value
firstName John
lastName Doe
age 50
eyeColor blue
fullName function() {return this.firstName + " " + this.lastName;}

Yöntemler, nesne özellikleri olarak depolanan fonksiyonlardır.

JavaScript Nesne Yöntemlerine Erişim


Bir nesne yöntemine aşağıdaki Sytnax ile erişirsiniz: 
objectName.methodName()

Genellikle fullName () öğesini person nesnesinin bir yöntemi olarak ve fullName özelliğini de bir özellik olarak tanımlarsınız.
FullName özelliği () ile çağrıldığında (fonksiyon olarak) yürütülür.
Bu örnek , bir kişi nesnesinin fullName () yöntemine erişir : 
    name = person.fullName();

FullName özelliğine () olmadan erişirseniz , fonksiyon tanımını döndürür :
Örnek: 
    name = person.fullName;

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Yerleşik Yöntemleri Kullanma


Bu örnek, toUpperCase () bir metin büyük harfe dönüştürmek için Dize nesnesinin yöntemini seçin: 
    var message = "Hello world!";
    var x = message.toUpperCase();

Yukarıdaki kod yürütüldükten sonra x değeri: 
    HELLO WORLD!

Bir Nesneye Yöntem Ekleme


Bir nesneye yeni bir yöntem eklemek kolaydır:
Örnek: 
    person.name = function () {
      return this.firstName + " " + this.lastName;
    };
