Nesne Türleri (Blueprints) (Classes)


Bazen aynı "tipte" birçok nesne oluşturmak için bir " taslak " lazım .
Bir "nesne türü" yaratmanın yolu, bir nesne yapıcı fonksiyonu kullanmaktır .
Yukarıdaki örnekte fonksiyon Person(), bir nesne yapıcı fonksiyondur.
Aynı türden nesneler, yapıcı fonksiyonları şu new anahtar sözcükle çağırarak oluşturulur : 
    var myFather = new Person("John", "Doe", 50, "blue");
    var myMother = new Person("Sally", "Rally", 48, "green");

This Anahtar Kelimesi


JavaScript'te, çağrılan şey this, koda "sahip" olan nesnedir.
this, Bir nesnede kullanıldığında değeri nesnenin kendisidir.
Bir yapıcı fonksiyonunda this'in bir değeri yoktur. Yeni nesnenin yerine geçer. this Yeni bir nesne oluşturulduğunda değeri yeni nesne olur.

Bir Nesneye Özellik Ekleme


Mevcut bir nesneye yeni bir özellik eklemek kolaydır:
Örnek: 
    myFather.nationality = "English";

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Özellik myFather'a eklenecek. myMother'a değil. (Başka hiçbir nesneye değil yani).

Bir Nesneye Yöntem Ekleme


Mevcut bir nesneye yeni bir yöntem eklemek kolaydır:
Örnek 
    myFather.name = function () {
      return this.firstName + " " + this.lastName;
    };

Özellik myFather'a eklenecek. myMother'a değil. (Başka hiçbir nesneye değil yani).

Yapıcıya Özellik Ekleme


Varolan bir nesneye yeni bir özellik eklediğiniz gibi, nesne oluşturucuya yeni bir özellik ekleyemezsiniz:
Örnek: 
    Person.nationality = "English";

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Çözümü ise şu şekildedir: 
    function Person(first, last, age, eyecolor) {
      this.firstName = first;
      this.lastName = last;
      this.age = age;
      this.eyeColor = eyecolor;
      this.nationality = "English";
    }

Bu şekilde nesne özellikleri varsayılan değerlere sahip olabilir.

Bir Yapıcıya Yöntem Ekleme


Örnek 
    function Person(first, last, age, eyecolor) {
      this.firstName = first;
      this.lastName = last;
      this.age = age;
      this.eyeColor = eyecolor;
      this.name = function() {return this.firstName + " " + this.lastName;};
    }

Varolan bir nesneye yeni bir yöntem eklediğiniz gibi, nesne oluşturucuya yeni bir yöntem ekleyemezsiniz.
Nesne yapıcısına yöntem ekleme, yapıcı fonksiyonu içinde yapılmalıdır:
Örnek 
    function Person(firstName, lastName, age, eyeColor) {
      this.firstName = firstName;
      this.lastName = lastName;
      this.age = age;
      this.eyeColor = eyeColor;
      this.changeName = function (name) {
        this.lastName = name;
      };
    }

ChangeName() işlevi, adın değerini kişinin lastName özelliğine atar. 
    Şimdi deneyebilirsiniz:

    myMother.changeName("Doe");

JavaScript "substituting" tarafından bahsetme şeklini kişi bilir this ile myMother .

Yerleşik JavaScript Oluşturucuları


JavaScript yerel nesneler için yerleşik yapıcılara sahiptir: 
    var x1 = new Object();    // A new Object object
    var x2 = new String();    // A new String object
    var x3 = new Number();    // A new Number object
    var x4 = new Boolean();   // A new Boolean object
    var x5 = new Array();     // A new Array object
    var x6 = new RegExp();    // A new RegExp object
    var x7 = new Function();  // A new Function object
    var x8 = new Date();      // A new Date object

Math()Nesne listesinde değil. Math global bir nesnedir. new anahtar kelime üzerinde kullanılamaz Math. 
    var x1 = {};            // new object
    var x2 = "";            // new primitive string
    var x3 = 0;             // new primitive number
    var x4 = false;         // new primitive boolean
    var x5 = [];            // new array object
    var x6 = /()/           // new regexp object
    var x7 = function(){};  // new function object

String Nesneleri


Normalde, strings ilkel olarak oluşturulur: var firstName = "John"
Ancak strings , new anahtar kelime kullanılarak nesne olarak da oluşturulabilir :var firstName = new String("John")
Stringlerin neden JS Dizeleri bölümünde nesne olarak oluşturulmaması gerektiğini öğrenin .

Sayı Nesneleri


Normalde sayılar ilkel olarak oluşturulur: var x = 123
Ancak sayılar, new anahtar kelime kullanılarak nesne olarak da oluşturulabilir :var x = new Number(123)

Boolean Nesneleri


Normalde, booleans ilkel olarak oluşturulur: var x = false
ncak booleans new anahtar kelimesini kullanarak nesne olarak da oluşturulabilir :var x = new Boolean(false)
