JavaScript Nesne Erişimcileri (Getters and Setters)


ECMAScript 5 (2009) Getter ve Setters'ı tanıttı.
Harfler ve ayarlayıcılar Nesne Erişimcilerini (Hesaplanan Özellikler) tanımlamanızı sağlar.

JavaScript Getter


Bu örnek, lang özelliğin get değerine ait bir özellik kullanır.
Örnek 
    // Nesne oluşturma
    var person = {
      firstName: "John",
      lastName : "Doe",
      language : "en",
      get lang() {
        return this.language;
      }
    };

    // Getter kullanarak nesneden verileri görüntüleme:
    document.getElementById("demo").innerHTML = person.lang;

JavaScript Setter


Bu örnek, language özelliğinin değerini ayarlamak için bir lang özelliği kullanır.
Örnek: 
    var person = {
      firstName: "John",
      lastName : "Doe",
      language : "",
      set lang(lang) {
        this.language = lang;
      }
    };

    // setter:
    person.lang = "en";

    // Display data object:
    document.getElementById("demo").innerHTML = person.language;

JavaScript Fonksiyon veya Getter?


Sizce bu iki örnek arasındaki farklar nelerdir?
Örnek 1: 
    var person = {
      firstName: "John",
      lastName : "Doe",
      fullName : function() {
        return this.firstName + " " + this.lastName;
      }
    };

    // Display object method:
    document.getElementById("demo").innerHTML = person.fullName();

Örnek 2: 
    var person = {
      firstName: "John",
      lastName : "Doe",
      get fullName() {
        return this.firstName + " " + this.lastName;
      }
    };

    // Display data object getter:
    document.getElementById("demo").innerHTML = person.fullName;

Örnek 1 fullName işlevine erişim: person.fullName ().
Örnek 2 fullName'a özellik olarak erişin: person.fullName.
İkinci örnek daha basit bir sözdizimi sağlar.

Veri kalitesi


JavaScript, alıcıları ve ayarlayıcıları hakkında daha iyi veri kalitesi mevcuttur.
Bu örnekte lang özelliğini kullanmak, özelliğin değerini language büyük harf olarak döndürür:
Örnek: 
    // Create an object:
    var person = {
      firstName: "John",
      lastName : "Doe",
      language : "en",
      get lang() {
        return this.language.toUpperCase();
      }
    };

    // Display data from the object using a getter:
    document.getElementById("demo").innerHTML = person.lang;

Bu örnekte, lang özelliği kullanmak, özelliğe bir büyük harf değeri depolar language: 
    var person = {
      firstName: "John",
      lastName : "Doe",
      language : "",
      set lang(lang) {
        this.language = lang.toUpperCase();
      }
    };

    // Set an object property using a setter:
    person.lang = "en";

    // Display data from the object:
    document.getElementById("demo").innerHTML = person.language;

Neden Getter ve Setter Kullanılmalı?


  • Daha basit bir sözdizimi sağlar
  • Özellikler ve yöntemler için eşit sözdizimine izin verir
  • Daha iyi veri kalitesi sağlayabilir
  • Sahne arkasındaki işleri yapmak için kullanışlıdır

    • Object.defineProperty()


    Object.defineProperty() yöntemi ayrıca, getter ve setterlara eklemek için kullanılabilir: 
        // Define object
    var obj = {counter : 0};

    // Define setters
    Object.defineProperty(obj, "reset", {
      get : function () {this.counter = 0;}
    });
    Object.defineProperty(obj, "increment", {
      get : function () {this.counter++;}
    });
    Object.defineProperty(obj, "decrement", {
      get : function () {this.counter--;}
    });
    Object.defineProperty(obj, "add", {
      set : function (value) {this.counter += value;}
    });
    Object.defineProperty(obj, "subtract", {
      set : function (value) {this.counter -= value;}
    });

    // Play with the counter:
    obj.reset;
    obj.add = 5;
    obj.subtract = 1;
    obj.increment;
    obj.decrement;
