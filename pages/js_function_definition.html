JavaScript Fonksiyon Tanımları


JavaScript fonksiyonları, function anahtar sözcüğü ile tanımlanır.
Bir fonksiyon declaration veya bir fonksiyon expression kullanabilirsiniz.

JavaScript Function Declarations


Function declared Syntax: 
    function functionName(parameters) {
     // kod bloğu çalışacak
    }

Bildirilen fonksiyonlar hemen yürütülmez. "Daha sonra kullanılmak üzere kaydedilirler" ve çağrıldıklarında daha sonra çalıştırılırlar.
Örnek: 
    function myFunction(a, b) {
      return a * b;
    }

Noktalı virgüller yürütülebilir JavaScript ifadelerini ayırmak için kullanılır.
Fonksiyon bildirimi yürütülebilir bir ifade olmadığından, bunu noktalı virgülle bitirmek yaygın değildir.

JavaScript Fonksiyon İfadeleri


Bir JavaScript işlevi bir fonksiyon kullanılarak da tanımlanabilir.
Bir fonksiyon ifadesi bir değişkende saklanabilir:
Örnek: 
    var x = function (a, b) {return a * b};

Bir fonksiyon ifadesi bir değişkende saklandıktan sonra, değişken bir fonksiyon olarak kullanılabilir:
Örnek: 
    var x = function (a, b) {return a * b};
    var z = x(4, 3);

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Yukarıdaki işlev aslında anonim bir fonksiyondur. (adsız bir fonksiyon)
Değişkenlerde saklanan fonksiyonların fonksiyon isimlerine ihtiyacı yoktur. Değişken adı kullanılarak her zaman çağrılır.

Yukarıdaki işlev, noktalı virgülle biter, çünkü yürütülebilir bir ifadenin bir parçasıdır.

JavaScript Yapıcı Fonksiyon ( The Function() Constructor )


Önceki örneklerde gördüğünüz gibi, JavaScript fonksiyonları function anahtar sözcüğü ile tanımlanır.
Fonksiyonlar ayrıca Function () adlı yerleşik bir JavaScript fonksiyon yapıcısıyla da tanımlanabilir.
Örnek: 
    var myFunction = new Function("a", "b", "return a * b");

    var x = myFunction(4, 3);

Aslında fonksiyon yapıcısını kullanmak zorunda değilsiniz. Şimdiki yapacağımız işlem yukarıdaki örneği yazmakla aynıdır:
Örnek: 
     var myFunction = function (a, b) {return a * b};

    var x = myFunction(4, 3);

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.



Çoğu zaman, yeni anahtar kelimeyi JavaScript'te kullanmaktan kaçınabilirsiniz.

JavaScript Fonksiyon Kaldırma


Kaldırma, JavaScript'in bildirimleri geçerli kapsamın üstüne taşıma varsayılan davranışıdır.
Kaldırma, değişken bildirimler ve fonksiyon bildirimleri için geçerlidir.
Bu nedenle, JavaScript fonksiyonları bildirilmeden önce çağrılabilir: 
    myFunction(5);

    function myFunction(y) {
      return y * y;
    }

Bir ifade kullanılarak tanımlanan fonksiyonlar kaldırılmaz.

JavaScript Kendini Çağıran Fonksiyonlar


Fonksiyon ifadeleri "kendi kendini çağıran" yapılarda olabilir.
Kendini çağıran bir ifade çağrılmadan otomatik olarak çağrılır.
İfadeyi () izliyorsa fonksiyon ifadeleri otomatik olarak yürütülür.
Bir fonksiyon bildirimini kendiniz çağıramazsınız.
Bir fonksiyon ifadesi olduğunu belirtmek için fonksiyonun etrafına parantez eklemeniz gerekir:
Örnek: 
    (function () {
      var x = "Hello!!";  // I will invoke myself
    })();

Yukarıdaki fonksiyon aslında anonim kendi kendini çağıran bir işlevdir (adsız fonksiyon).

Fonksiyonlar Değer Olarak Kullanılabilir


JavaScript fonksiyonları değer olarak kullanılabilir:
Örnek: 
    function myFunction(a, b) {
      return a * b;
    }

JavaScript fonksiyonları ifadelerde kullanılabilir: 
    function myFunction(a, b) {
      return a * b;
    }

    var x = myFunction(4, 3) * 2;

Fonksiyonlar Nesnelerdir


JavaScript'teki typeof operatörü fonksiyonlar için "function" döndürür.
Ancak, JavaScript fonksiyonları en iyi nesne olarak tanımlanabilir.
JavaScript fonksiyonlarının hem özellikleri hem de yöntemleri vardır.
Arguments.length özelliği, fonksiyon çağrıldığında alınan bağımsız değişkenlerin sayısını döndürür:
Örnek: 
    function myFunction(a, b) {
      return arguments.length;
    }

ToString () yöntemi fonksiyonu bir string olarak döndürür:
Örnek: 
    function myFunction(a, b) {
      return a * b;
    }

    var txt = myFunction.toString();

Arrow Fonksiyonları


Arrow fonksiyonları, fonksiyon ifadeleri yazmak için kısa bir sözdizimi sağlar.
Fonksiyon anahtar sözcüğüne, return anahtar sözcüğüne ve süslü parantezlere ihtiyacınız yoktur.
Örnek: 
    // ES5
    var x = function(x, y) {
      return x * y;
    }

    // ES6
    const x = (x, y) => x * y;

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Arrow fonksiyonlarının kendilerine ait bir yönü yoktur. Nesne yöntemlerini tanımlamak için uygun değildirler.
Arrow fonksiyonları kaldırılmaz. Kullanılmadan önce tanımlanmalıdırlar.
Const kullanmak var kullanmaktan daha güvenlidir, çünkü bir işlev ifadesi her zaman sabit değerdir.
Return anahtar sözcüğünü ve süslü ayraçları yalnızca işlev tek bir deyimse atlayabilirsiniz. Bu nedenle, onları her zaman tutmak iyi bir alışkanlık olabilir:
Örnek: 
    const x = (x, y) => { return x * y };
