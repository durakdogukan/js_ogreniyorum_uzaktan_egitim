JavaScript Kapanışları


JavaScript değişkenleri yerel veya global kapsama ait olabilir.
Global değişkenler kapanışlarla yerel (özel) yapılabilir.

Global Değişkenleri


Bir fonksiyon, fonksiyonun içinde tanımlanan tüm değişkenlere şu şekilde erişebilir:
Örnek: 
    function myFunction() {
      var a = 4;
      return a * a;
    }

Ancak bir fonksiyon, fonksiyon dışında tanımlanan değişkenlere de şu şekilde erişebilir:
Örnek: 
    var a = 4;
    function myFunction() {
      return a * a;
    }

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Son örnekte, a global bir değişkendir.
Bir web sayfasında, global değişkenler pencere nesnesine aittir.
Genel değişkenler sayfadaki (ve penceredeki) tüm komut dosyaları tarafından kullanılabilir (ve değiştirilebilir).
İlk örnekte, a yerel bir değişkendir.
Bir yerel değişken sadece tanımlandığı fonksiyonun içinde kullanılabilir. Diğer fonksiyonlardan ve diğer komut dosyası kodlarından gizlenir.
Aynı ada sahip genel ve yerel değişkenler farklı değişkenlerdir. Birini değiştirmek diğerini değiştirmez.

Değişkenin Ömrü


Genel değişkenler, başka bir sayfaya gittiğinizde veya pencereyi kapattığınızda olduğu gibi sayfa atılana kadar yaşar.
Yerel değişkenlerin kısa ömürleri vardır. Bunlar fonksiyon çağrıldığında oluşturulur ve fonksiyon tamamlandığında silinir.

A Counter Dilemma


Bir şeyi saymak için bir değişken kullanmak istediğinizi ve bu sayacın tüm işlevler için kullanılabilir olmasını istediğinizi varsayalım.
Genel bir değişken ve sayacı artırmak için bir işlev kullanabilirsiniz:
Örnek: 
    // Sayaç başlatılır
    var counter = 0;
    // Sayacı arttırma işlevi
    function add() {
      counter += 1;
    }

    // add() 3 kere çağırıldı
    add();
    add();
    add();

    // Sayaç artık 3

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Yukarıdaki çözümde bir sorun var: Sayfadaki herhangi bir kod, add () öğesini çağırmadan sayacı değiştirebilir.
Sayaç, diğer kodların değiştirilmesini önlemek için add () fonksiyonunda yerel olmalıdır:
Örnek 
    // Sayaç başlatılır
    var counter = 0;

    // Sayacı arttırma fonksiyonu
    function add() {
      var counter = 0;
      counter += 1;
    }

    // add() 3 kere çağırıldı
    add();
    add();
    add();

    //Sayaç artık 3 olmalı fakat sayaç 0 olur...

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


İşe yaramadı çünkü yerel sayaç yerine global sayacı görüntülüyoruz.
Global sayacı kaldırabilir ve fonksiyonun geri döndürmesine izin vererek yerel sayaca erişebiliriz:
Örnek 
    // Sayacı arttırma fonksiyonu
    function add() {
      var counter = 0;
      counter += 1;
      return counter;
    }

    // add() 3 kere çağırıldı
    add();
    add();
    add();

    //Sayaç artık 3 olmalı fakat sayaç 1 olur...

Fonksiyonu her çağırdığımızda yerel sayacı sıfırladığımız için çalışmadı.
Bir JavaScript iç fonksiyonu bunu çözebilir.

JavaScript İç İçe Fonksiyonlar


Tüm fonksiyonların global kapsama erişimi vardır.
Aslında, JavaScript'te, tüm fonksiyonların kendilerinin "üstündeki" kapsama erişimi vardır.
JavaScript iç içe fonksiyonları destekler.
Bu örnekte iç fonksiyon plus (), üst fonksiyondaki sayaç değişkenine erişime sahiptir:
Örnek: 
    function add() {
      var counter = 0;
      function plus() {counter += 1;}
      plus();
      return counter;
    }

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Dışarıdan plus () fonksiyonuna erişebilseydik, bu karşı ikilemi çözebilirdi.
Ayrıca counter = 0'ı yalnızca bir kez yürütmenin bir yolunu bulmamız gerekir.
Bir closure a ihtiyacımız var.

JavaScript Closures


Kendini çağıran fonksiyonları hatırlıyor musunuz? Bu fonksiyon ne işe yarıyor sizce? 
    var add = (function () {
      var counter = 0;
      return function () {counter += 1; return counter}
    })();

    add();
    add();
    add();

    // Sayaç 3

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Örneğin Açıklaması


Add değişkeni, kendini çağırma fonksiyonunun dönüş değeri olarak atanır.
Kendini çağırma fonksiyonu yalnızca bir kez çalışır. Sayacı sıfıra (0) ayarlar ve bir fonksiyon ifadesi döndürür.
Bu şekilde ekleme fonksiyonu olur. "Harika" kısmı, ana kapsamdaki sayaca erişebilmesidir.
Buna JavaScript closures adı verilir. Bir fonksiyonun "özel" değişkenlere sahip olmasını mümkün kılar.
Sayaç, anonim fonksiyonun kapsamı tarafından korunur ve yalnızca add fonksiyonu kullanılarak değiştirilebilir.
