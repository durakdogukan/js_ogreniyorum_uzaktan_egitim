JavaScript Tip Dönüşümleri


Number () bir sayıya, String () bir String'e, Boolean () bir Boolean'a dönüştürür.

JavaScript Veri Türleri


JavaScript'te değer içerebilen 5 farklı veri türü vardır:
  • string
  • number
  • boolean
  • object
  • function

    • 6 tür nesne vardır:
  • Object
  • Date
  • Array
  • String
  • Number
  • Boolean

    • Ve değer içeremeyen 2 veri türü daha vardır:
  • null
  • undefined

    • Typeof Operatörü


    Bir JavaScript değişkeninin veri türünü bulmak için typeof operatörünü kullanabilirsiniz. 
        typeof "John"                 // "string" döndürür
    typeof 3.14                   // "number" döndürür
    typeof NaN                    // "number" döndürür
    typeof false                  // "boolean" döndürür
    typeof [1,2,3,4]              // "object" döndürür
    typeof {name:'John', age:34}  // "object" döndürür
    typeof new Date()             // "object" döndürür
    typeof function () {}         // "function" döndürür
    typeof myCar                  // "undefined" * döndürür
    typeof null                   // "object" döndürür

    Typeof Veri Türü


    Typeof operatörü bir değişken değildir. Bir operatör. Operatörlerin (+ - * /) herhangi bir veri türü yoktur.
    Ancak, typeof operatörü her zaman bir dize döndürür (işlenenin türünü içerir).

    Constructor özelliği


    Constructor özelliği, tüm JavaScript değişkenleri için yapıcı işlevini döndürür. 
        "John".constructor                // Returns function String()  {[native code]}
    (3.14).constructor                // Returns function Number()  {[native code]}
    false.constructor                 // Returns function Boolean() {[native code]}
    [1,2,3,4].constructor             // Returns function Array()   {[native code]}
    {name:'John',age:34}.constructor  // Returns function Object()  {[native code]}
    new Date().constructor            // Returns function Date()    {[native code]}
    function () {}.constructor        // Returns function Function(){[native code]}

    JavaScript Tip Dönüşümleri


    JavaScript değişkenleri yeni bir değişkene ve başka bir veri türüne dönüştürülebilir:
  • JavaScript fonksiyonu kullanarak
  • JavaScript'in kendisi tarafından otomatik olarak

    • Sayıları Stringlere Dönüştürme


    Global yöntem String () sayıları stringe dönüştürebilir.
    Herhangi bir sayı, değişmez değer, değişken veya ifadede kullanılabilir: 
        String(x)         // returns a string from a number variable x
    String(123)       // returns a string from a number literal 123
    String(100 + 23)  // returns a string from a number from an expression

    Örneği incelemek için tıklayın
     
     

    NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


    ToString () yönteminin Number yöntemi de aynısını yapar. 
        x.toString()
    (123).toString()
    (100 + 23).toString()

    Örneği incelemek için tıklayın
     
     

    NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


    Method Açıklama
    toExponential() Üstel gösterim kullanılarak yuvarlanmış ve yazılmış bir sayı içeren bir dize döndürür.
    toFixed() Belirtilen sayıda ondalık sayı ile yuvarlanmış ve yazılmış bir dize döndürür.
    toPrecision() Belirtilen sayıda ondalık sayı ile yuvarlanmış ve yazılmış bir dize döndürür.

    Booleanları Dizelere Dönüştürme


    Global yöntem String (), boolean'ları dizelere çevirebilir. 
        String(false)      // "false" döndürür
    String(true)       // "true" döndürür

    ToString () yöntemi Boole yöntemi de bunu yapar. 
        false.toString()   // "false" döndürür
    true.toString()    // "true" döndürür

    Tarihleri Stringlere Dönüştürme


    Genel yöntem String (), tarihleri stringlere dönüştürebilir. 
        String(Date())  // returns "Thu Jul 17 2014 15:38:19 GMT+0200 (W. Europe Daylight Time)"

    Date yöntemi toString () de aynı şeyi yapar. 
        Date().toString()  // returns "Thu Jul 17 2014 15:38:19 GMT+0200 (W. Europe Daylight Time)"

    Method Açıklama
    getDate() Günü sayı olarak al (1-31)
    getDay() Hafta içi bir numara al (0-6)
    getFullYear() Dört haneli yılı al (yyyy)
    getHours () Saati al (0-23)
    getMilliseconds () Milisaniye al (0-999)
    getMinutes() Dakikayı al (0-59)
    getMonth () Ayı al (0-11)
    getSeconds () Saniyeyi al (0-59)
    getTime() Zamanı al (1 Ocak 1970'den bu yana milisaniye)

    Stringleri Sayılara Dönüştürme


    Global yöntem Number (), dizeleri sayılara dönüştürebilir.
    Sayı içeren dizeler ("3.14" gibi) sayılara dönüştürülür (3.14 gibi).
    Boş dizeler 0'a dönüşür.
    Başka bir şey NaN (Sayı Değil) biçimine dönüşür. 
        Number("3.14")    // 3.14 döndürür
    Number(" ")       // 0 döndürür
    Number("")        // 0 döndürür
    Number("99 88")   // NaN döndürür

    Method Açıklama
    parseFloat() Bir string'i ayrıştırır ve bir kayan nokta sayısı döndürür
    parseInt() Bir string'i ayrıştırır ve bir tamsayı döndürür

    Unary + Operatörü


    Unary + operatörü bir değişkeni sayıya dönüştürmek için kullanılabilir: 
        var y = "5";      // y string
    var x = + y;      // x number

    Booleanları Sayılara Dönüştürme


    Number () global yöntemi booleanları sayılara dönüştürebilir. 
        Number(false)     // 0 döndürür
    Number(true)      // 1 döndürür

    Tarihleri Sayılara Dönüştürme


    number () genel yöntemi, tarihleri sayılara dönüştürmek için kullanılabilir. 
        d = new Date();
    Number(d)          // 1404568027739 döndürür

    GetTime () tarih yöntemi aynı şeyi yapar. 
        d = new Date();
    d.getTime()        // 1404568027739 döndürür

    Otomatik Tip Dönüştürme


    JavaScript "yanlış" bir veri türünde çalışmaya çalıştığında, değeri "doğru" bir türe dönüştürmeye çalışır.
    Sonuç her zaman beklediğiniz gibi değildir: 
        5 + null    // returns 5         because null is converted to 0
    "5" + null  // returns "5null"   because null is converted to "null"
    "5" + 2     // returns "52"      because 2 is converted to "2"
    "5" - 2     // returns 3         because "5" is converted to 5
    "5" * "2"   // returns 10        because "5" and "2" are converted to 5 and 2

    Otomatik String'e Dönüştürme


    Bir nesneyi veya değişkeni "çıktıya" almaya çalıştığınızda JavaScript otomatik olarak değişkenin toString () fonksiyonunu çağırır: 
        document.getElementById("demo").innerHTML = myVar;
    // if myVar = {name:"Fjohn"}  // toString converts to "[object Object]"
    // if myVar = [1,2,3,4]       // toString converts to "1,2,3,4"
    // if myVar = new Date()      // toString converts to "Fri Jul 18 2014 09:08:55 GMT+0200"

    Sayılar ve booleanlar da dönüştürülür, ancak bu çok görünür değildir: 
        // if myVar = 123             // toString converts to "123"
    // if myVar = true            // toString converts to "true"
    // if myVar = false           // toString converts to "false"
