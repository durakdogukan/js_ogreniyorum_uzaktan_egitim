JavaScript Fonksiyon Parametreleri


JavaScript fonksiyonu, parametre değerleri (bağımsız değişkenler) üzerinde herhangi bir denetim gerçekleştirmez.

JavaScript Fonksiyon Parametreleri ve Bağımsız Değişkenler


    function functionName(parameter1, parameter2, parameter3) {
      // kod bloğu çalışacak
    }

Fonksiyon parametreleri, fonksiyon tanımında listelenen adlardır.
Fonksiyon bağımsız değişkenleri, fonksiyona iletilen (ve bu fonksiyon tarafından alınan) gerçek değerlerdir.

JavaScript Parametre Kuralları


JavaScript fonksiyon tanımları, parametreler için veri türlerini belirtmez.
JavaScript fonksiyonları, iletilen bağımsız değişkenlerde tür denetimi gerçekleştirmez.
JavaScript fonksiyonları, alınan bağımsız değişkenlerin sayısını kontrol etmez.

JavaScript Parametre Varsayılanları


Bir fonksiyon eksik argümanlarla (bildirilenden daha az) çağrılırsa, eksik değerler şu şekilde ayarlanır: undefined
Bazen bu kabul edilebilir, ancak bazen parametreye varsayılan bir değer atamak daha iyidir:
Örnek: 
    function myFunction(x, y) {
      if (y === undefined) {
        y = 0;
      }
    }

    ECMAScript 2015, fonksiyon bildiriminde varsayılan parametre değerlerine izin verir:

    function (a=1, b=1) {
      // fonksiyon kodu
    }

JavaScript Arguments Nesnesi


JavaScript fonksiyonlarında arguments nesnesi adı verilen yerleşik bir nesne vardır.
Bağımsız değişken nesnesi, fonksiyon çağrıldığında (çağrıldığında) kullanılan bir dizi bağımsız değişken içerir.
Bu şekilde, bir sayı listesindeki en yüksek değeri bulmak (örneğin) için bir foknsiyonu kullanabilirsiniz:
Örnek: 
    x = findMax(1, 123, 500, 115, 44, 88);

    function findMax() {
      var i;
      var max = -Infinity;
      for (i = 0; i < arguments.length; i++) {
        if (arguments[i] > max) {
          max = arguments[i];
        }
      }
      return max;
    }

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Veya tüm giriş değerlerini toplamak için bir fonksiyon oluşturun:
Örnek: 
    x = sumAll(1, 123, 500, 115, 44, 88);

    function sumAll() {
      var i;
      var sum = 0;
      for (i = 0; i < arguments.length; i++) {
        sum += arguments[i];
      }
      return sum;
    }

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Bir fonksiyon çok fazla bağımsız değişkenle (bildirilenden fazla) çağrılırsa, bu bağımsız değişkenlere arguments nesnesi kullanılarak erişilebilir.

Arguments Değerden Geçirilir


Fonksiyon çağrısındaki parametreler, fonksiyonun bağımsız değişkenleridir.
JavaScript bağımsız değişkenleri değere göre iletilir: Fonksiyon, bağımsız değişkenin konumlarını değil, yalnızca değerleri tanır.
Bir işlev bir bağımsız değişkenin değerini değiştirirse, parametrenin orijinal değerini değiştirmez.
Değişkenlerde yapılan değişiklikler işlevin dışında görünmez (yansıtılmaz).

Nesneler Referans ile Geçirilir


JavaScript'te nesne başvuruları değerlerdir.
Bu nedenle, nesneler referans olarak iletildikleri gibi davranacaktır:
Bir fonksiyon bir nesne özelliğini değiştirirse, orijinal değeri değiştirir.
Nesne özelliklerinde yapılan değişiklikler fonksiyonun dışında görünür (yansıtılır).
