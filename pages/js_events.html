JavaScript Olaylar (Events)


HTML olayları, HTML öğelerinde meydana gelen "şeyler" dir.
HTML sayfalarında JavaScript kullanıldığında, JavaScript bu olaylara "tepki verebilir".

HTML Olayları


HTML olayı, tarayıcının yaptığı veya kullanıcının yaptığı bir şey olabilir.
HTML olaylarına bazı örnekler:
  • Bir HTML web sayfasının yüklenmesi tamamlandı
  • Bir HTML giriş alanı değiştirildi
  • Bir HTML düğmesi tıklandı

    • Çoğu zaman, olaylar meydana geldiğinde, bir şeyler yapmak isteyebilirsiniz.
    JavaScript, olaylar algılandığında kod yürütmenizi sağlar.
    HTML, HTML kodlarına JavaScript kodlu olay işleyici niteliklerinin eklenmesine olanak tanır.
    Tek tırnak işaretleri ile:
    <element event='some JavaScript'>

    Çift tırnak işaretleri ile:
    <element event="some JavaScript ">

    Aşağıdaki örnekte, bir button öğesine bir onclick niteliği (kodlu) eklenmiştir:
    <button onclick="document.getElementById('demo').innerHTML = Date()">The time is?</button>

    NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


    Yukarıdaki örnekte, JavaScript kodu öğenin içeriğini id = "demo" ile değiştirir.
    Sonraki yapacağımız örnekte, kod kendi öğesinin içeriğini değiştirir (this.innerHTML kullanarak):
    <button onclick="this.innerHTML = Date()">The time is?</button>

    Örneği incelemek için tıklayın
     
     

    NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


    <button onclick="displayDate()">The time is?</button>

    Ortak HTML Etkinlikleri


    Aşağıda bazı yaygın HTML etkinliklerinin bir listesi verilmiştir:
    Olay Açıklama
    onchange Bir HTML öğesi değiştirildi
    onclick Kullanıcı bir HTML öğesini tıklar
    onmouseover Kullanıcı fareyi bir HTML öğesinin üzerine taşır
    onmouseout Kullanıcı fareyi bir HTML öğesinden uzaklaştırır
    onkeydown Kullanıcı bir klavye tuşuna basar
    onload Tarayıcı sayfayı yüklemeyi tamamladı

    JavaScript ne yapabilir?


  • Bir sayfa her yüklendiğinde yapılması gerekenler
  • Sayfa kapalıyken yapılması gerekenler
  • Kullanıcı bir düğmeyi tıkladığında gerçekleştirilmesi gereken eylem
  • Bir kullanıcı veri girdiğinde doğrulanması gereken içerik
  • Ve dahası ...

    • JavaScript'in etkinliklerle çalışmasına izin vermek için birçok farklı yöntem kullanılabilir:
  • HTML olay özellikleri doğrudan JavaScript kodunu yürütebilir
  • HTML olay özellikleri JavaScript fonksiyonlarını çağırabilir
  • HTML öğelerine kendi olay işleyici fonksiyonlarını atayabilirsiniz
  • Etkinliklerin gönderilmesini veya işlenmesini engelleyebilirsiniz
  • Ve dahası ...
