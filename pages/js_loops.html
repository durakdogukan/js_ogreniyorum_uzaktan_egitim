JavaScript Döngüler


Her seferinde farklı bir değerle aynı kodu tekrar tekrar çalıştırmak istiyorsanız döngüler kullanışlıdır.
Genellikle dizilerle çalışırken durum budur: 
Bunu yazmak yerine:

    text += cars[0] + "\n";
    text += cars[1] + "\n";
    text += cars[2] + "\n";
    text += cars[3] + "\n";
    text += cars[4] + "\n";
    text += cars[5] + "\n";


Bunu yazabilirsin:

    var i;
    for (i = 0; i < cars.length; i++) {
      text += cars[i] + "\n";
    }

Farklı Döngü Çeşitleri


JavaScript farklı döngü türlerini destekler:
  • for - bir kod bloğundan birkaç kez geçer
  • for / in - nesnenin özellikleri arasında dolaşır
  • for / of - yinelenebilir bir nesnenin değerleri arasında geçiş yapar
  • while - belirtilen bir koşul geçerliyken bir kod bloğundan geçer
  • do / while - ayrıca belirtilen bir koşul geçerliyken bir kod bloğunda da dolaşır

    • For Döngüsü


    For döngüsü aşağıdaki sözdizimine sahiptir:
    Syntax 
        for (statement 1; statement 2; statement 3) {
      // kod bloğu çalışacaktır.
    }


        İfade 1, kod bloğunun yürütülmesinden önce (bir kez) yürütülür.
    İfade 2, kod bloğunu yürütme koşulunu tanımlar.
    İfade 3, kod bloğu yürütüldükten sonra (her seferinde) yürütülür.

    Örnek 
        for (i = 0; i < 5; i++) {
    text += "The number is " + i + "\n";
    }

    Yukarıdaki örnekte şunları okuyabilirsiniz:
    İfade 1 döngü başlamadan önce bir değişken ayarlar (var i = 0).
    İfade 2, döngünün çalışması için koşulu tanımlar (i 5'ten küçük olmalıdır).
    İfade 3, döngüdeki kod bloğu her yürütüldüğünde bir değeri (i ++) artırır.

    İfade 1


    Normalde döngüde kullanılan değişkeni başlatmak için 1. ifadeyi kullanırsınız (i = 0).
    Bu her zaman böyle değildir, JavaScript umursamaz. İfade 1 isteğe bağlıdır.
    1. ifadede birçok değeri başlatabilirsiniz (virgülle ayrılmış olarak): 
        for (i = 0, len = cars.length, text = ""; i < len; i++) {
     text += cars[i] + "
    ";
    }

    İfade 1'i atlayabilirsiniz (değerleriniz döngü başlamadan önce ayarlandığında olduğu gibi): 
        var i = 2;
    var len = cars.length;
    var text = "";
    for (; i < len; i++) {
      text += cars[i] + "\n";
    }

    İfade 2


    Genellikle ifade 2, ilk değişkenin durumunu değerlendirmek için kullanılır.
    Bu her zaman böyle değildir, JavaScript umursamaz. Bildirim 2 de isteğe bağlıdır.
    İfade 2 true değerini döndürürse, döngü yeniden başlar, false döndürürse döngü sona erer.

    İfade 3


    Genellikle ifade 3, ilk değişkenin değerini artırır.
    Bu her zaman geçerli değildir, JavaScript umursamaz ve ifade 3 isteğe bağlıdır.
    İfade 3, negatif artış (i--), pozitif artış (i = i + 15) veya başka bir şey yapabilir.
    İfade 3 de atlanabilir (değerlerinizi döngü içinde artırdığınızda olduğu gibi): 
        var i = 0;
    var len = cars.length;
    for (; i < len; ) {
      text += cars[i] + "\n";
      i++;
    }

    JavaScript For/Of Döngüsü


    For/Of ifadesi için JavaScript yinelenebilir bir nesnenin değerleri arasında dolaşır
    for / of, Diziler, Dizeler, Haritalar, Düğüm Listeleri ve daha fazlası gibi yinelenebilir veri yapıları arasında geçiş yapmanızı sağlar.
    For / of loop aşağıdaki sözdizimine sahiptir: 
        for (variable of iterable) {
      // kod bloğu çalışacaktır
    }

    variable - Her yineleme için değişkene sonraki özelliğin değeri atanır. Değişken const, let veya var ile bildirilebilir.
    iterable - Yinelenebilir özelliklere sahip bir nesne.

    Bir Dizi Üzerinde Döngü


        var cars = ['BMW', 'Volvo', 'Mini'];
    var x;
    for (x of cars) {
      document.write(x + "\n");
    }

    JavaScript While Döngüsü


    Döngüler, belirtilen bir koşul geçerli olduğu sürece bir kod bloğu yürütebilir.
    While döngüsü, belirtilen bir koşul geçerli olduğu sürece bir kod bloğundan geçer.
    Syntax 
        while (condition) {
      // kod bloğu çalışacak
    }

    Örnek
    Aşağıdaki örnekte, bir değişken (i) 10'dan az olduğu sürece döngüdeki kod tekrar tekrar çalışacaktır: 
        while (i < 10) {
      text += "The number is " + i;
      i++;
    }

    JavaScript Do / While Döngüsü


    Do / while döngüsü, while döngüsünün bir çeşididir. Bu döngü, koşulun doğru olup olmadığını kontrol etmeden önce kod bloğunu bir kez yürütür, ardından koşul doğru olduğu sürece döngüyü tekrarlar.
    Syntax 
        do {
      // kod bloğu çalışacak
    }
    while (condition);

    Örnek
    Aşağıdaki örnekte bir do / while döngüsü kullanılmıştır. Kod bloğu, koşul test edilmeden önce yürütüldüğünden, koşul yanlış olsa bile her zaman en az bir kez yürütülür: 
        do {
      text += "The number is " + i;
      i++;
    }
    while (i < 10);

    Örneği incelemek için tıklayın
     
     

    NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


    Koşulda kullanılan değişkeni artırmayı unutmayın, aksi takdirde döngü asla bitmez!

    JavaScript For ve While Karşılaştırması


    For döngüsüyle ilgili önceki bölümü okuduysanız, while döngüsünün for döngüsü ile hemen hemen aynı olduğunu göreceksiniz, ifade 1 ve ifade 3 atlanmıştır.
    Bu örnekteki döngü, cars dizisinden araç adlarını toplamak için bir for döngüsü kullanır: 
        var cars = ["BMW", "Volvo", "Saab", "Ford"];
    var i = 0;
    var text = "";

    for (;cars[i];) {
      text += cars[i] + "\n";
      i++;
    }
