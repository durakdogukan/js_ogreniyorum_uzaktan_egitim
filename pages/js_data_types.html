JavaScript Veri Tipleri


JavaScript değişkenleri birçok veri türüne sahip olabilir: sayılar, dizeler, nesneler ve daha fazlası: 
    var length = 16;                               // Number
    var lastName = "Johnson";                      // String
    var x = {firstName:"John", lastName:"Doe"};    // Object

Veri Türleri Kavramı


Programlamada veri türleri önemli bir kavramdır.
Değişkenler üzerinde çalışabilmek için tür hakkında bir şeyler bilmek önemlidir.
Veri türleri olmadan bir bilgisayar bunu güvenli bir şekilde çözemez: 
    var x = 16 + "Volvo";

On altıya "Volvo" eklemek mantıklı mı? Bir hata mı üretecek yoksa bir sonuç üretecek mi?
JavaScript yukarıdaki örneği şu şekilde ele alacaktır: 
    var x = "16" + "Volvo";

Bir sayı ve dize eklerken, JavaScript numarayı string olarak görür.

    var x = 16 + "Volvo";

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz. 


    var x = "Volvo" + 16;

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Bir sayı ve string eklerken, JavaScript numarayı string olarak görür.... 
    var x = 16 + 4 + "Volvo";

    Sonuç: 20Volvo

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz. 


    var x = "Volvo" + 16 + 4;

    Sonuç: Volvo164

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


İlk örnekte, JavaScript 16 ve 4'ü "Volvo" ya ulaşana kadar sayı olarak ele alır.
İkinci örnekte, ilk işlenen bir dize olduğundan, tüm işlenenler string olarak ele alınır.

JavaScript Türleri Dinamiktir



JavaScript'in dinamik türleri vardır. Bu, aynı değişkenin farklı veri türlerini tutmak için kullanılabileceği anlamına gelir: 
    var x;           // Now x is undefined
    x = 5;           // Now x is a Number
    x = "John";      // Now x is a String

JavaScript Stringler


Bir string (veya bir metin dizesi) "John Doe" gibi bir dizi karakterdir.
Stringler tırnak işaretleri ile yazılır. Tek veya çift tırnak kullanabilirsiniz: 
    var carName1 = "Volvo XC60";   // Çift tırnak kullanılıyor.
    var carName2 = 'Volvo XC60';   // Tek tırnak kullanılıyor.

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Stringi çevreleyen tırnak işaretleriyle eşleşmedikleri sürece, bir stringin içindeki tırnak işaretlerini kullanabilirsiniz: 
    var answer1 = "It's alright";             // Çift tırnak içinde tek tırnak
    var answer2 = "He is called 'Johnny'";    // Çift tırnak içinde tek tırnaklar
    var answer3 = 'He is called "Johnny"';    // Tek tırnak içinde çift tırnak

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


JavaScript Sayılar


JavaScript'in yalnızca bir tür numarası vardır.
Sayılar ondalıklı veya ondalıksız yazılabilir: 
    var x1 = 34.00;     // Ondalık sayılarla yazılmıştır
    var x2 = 34;        // Ondalık sayılarla yazılmamıştır

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Ekstra büyük veya ekstra küçük sayılar bilimsel (üstel) gösterimle yazılabilir: 
    var y = 123e5;      // 12300000
    var z = 123e-5;     // 0.00123

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


JavaScript Boolean


Booleanların yalnızca iki değeri olabilir: doğru (true) veya yanlış (false). 
    var x = 5;
    var y = 5;
    var z = 6;
    (x == y)       // True döndürecek.
    (x == z)       // False döndürecek.

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Boolean genellikle koşullu testlerde kullanılır.

JavaScript Diziler


JavaScript dizileri köşeli parantez ile yazılır.
Dizi öğeleri virgülle ayrılır.
Aşağıdaki kod, üç öğe (araba adları) içeren otomobil adlı bir diziyi bildirir (oluşturur): 
    var cars = ["Saab", "Volvo", "BMW"];

JavaScript Operatör türü


Bir JavaScript değişkeninin türünü bulmak için JavaScript typeof operatörünü kullanabilirsiniz.
Typeof operatörü bir değişkenin veya ifadenin türünü döndürür: 
    typeof ""             // "string" döndürür
    typeof "John"         // "string" döndürür
    typeof "John Doe"     // "string" döndürür

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz. 


    typeof 0              // "number" döndürür
    typeof 314            // "number" döndürür
    typeof 3.14           // "number" döndürür
    typeof (3)            // "number" döndürür
    typeof (3 + 4)        // "number" döndürür

JavaScript NULL


JavaScript'te null "hiçbir şey" değildir. Var olmayan bir şey olması gerekiyordu.
Ne yazık ki, JavaScript'te null veri türü bir nesnedir.
Bir nesneyi null değerine ayarlayarak boşaltabilirsiniz: 
    var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
    person = null;    // Değer null, ama tipi hala bir object

Örneği incelemek için tıklayın
 
 

NOT : Kodu incelemek için örnek sayfasındayken CTRL+U tuşuna basınız. Karşınıza çıkan ekrandan kodları inceleyebilirsiniz.


Bir nesneyi tanımsız olarak ayarlayarak da boşaltabilirsiniz: 
    var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
    person = undefined;   // Şimdi hem değer hem de tür tanımsız

Örneği incelemek için tıklayın

Tanımsız ve Sıfır Arasındaki Fark


undefined ve null değerleri eşit, ancak türleri farklı: 
    typeof undefined           // undefined
    typeof null                // object
    null === undefined         // false
    null == undefined          // true
