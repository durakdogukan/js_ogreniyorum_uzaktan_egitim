JavaScript Fonksiyon Çağırma


Call () yöntemiyle, farklı nesnelerde kullanılabilecek bir yöntem yazabilirsiniz.

Tüm Fonksiyonlar Yöntemdir


JavaScript'te tüm fonksiyonlar nesne yöntemleridir.
Bir fonksiyon bir JavaScript nesnesinin yöntemi değilse, genel nesnenin bir fonksiyonudur (bkz. Önceki bölüm).
Aşağıdaki örnek, firstName, lastName, fullName olmak üzere 3 özelliğe sahip bir nesne oluşturur.
Örnek: 
    var person = {
      firstName:"John",
      lastName: "Doe",
      fullName: function () {
        return this.firstName + " " + this.lastName;
      }
    }
    person.fullName();   // "John Doe" döndürür

This Anahtar Kelimesi


Bir fonksiyon tanımında, bu fonksiyonun "sahibini" ifade eder.
Yukarıdaki örnekte, bu, fullName fonksiyonuna "sahip" olan kişi nesnesidir.
Başka bir deyişle, this.firstName bu nesnenin firstName özelliği anlamına gelir.

JavaScript call() Yöntemi


Call () yöntemi, önceden tanımlanmış bir JavaScript yöntemidir.
Bir bağımsız değişken (parametre) olarak sahip nesnesine sahip bir yöntemi çağırmak (çağırmak) için kullanılabilir.
Aşağıdaki örnekde person1 üzerinde kullanarak kişinin fullName yöntemini çağırır: 
    var person = {
      fullName: function() {
        return this.firstName + " " + this.lastName;
      }
    }
    var person1 = {
      firstName:"John",
      lastName: "Doe"
    }
    var person2 = {
      firstName:"Mary",
      lastName: "Doe"
    }
    person.fullName.call(person1);  // "John Doe" döndürür

Aşağıdaki örnek ise person2 üzerinde kullanarak kişinin fullName yöntemini çağırır: 
    var person = {
      fullName: function() {
        return this.firstName + " " + this.lastName;
      }
    }
    var person1 = {
      firstName:"John",
      lastName: "Doe"
    }
    var person2 = {
      firstName:"Mary",
      lastName: "Doe"
    }
    person.fullName.call(person2);  // "Mary Doe" döndürür

Arguments ile call () Yöntemi


Call () yöntemi arguments kabul edebilir:
Örnek: 
    var person = {
      fullName: function(city, country) {
        return this.firstName + " " + this.lastName + "," + city + "," + country;
      }
    }
    var person1 = {
      firstName:"John",
      lastName: "Doe"
    }
    person.fullName.call(person1, "Oslo", "Norway");
